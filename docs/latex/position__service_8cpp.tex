Description \+:

This node is a random position server. It responds with a random position (x,y,theta), which is used as a target to be reached. The random number for (x,y) position is between \mbox{[}min, max\mbox{]} that can be set in the request. The \mbox{[}min, max\mbox{]} values of random number for (theta) is initially set and fixed \mbox{[}-\/3.\+14, 3.\+14\mbox{]}
